# ICTU Community

ICTU Community is now configured as a Flutter mobile project using Dart.

## Tech Stack

- Frontend: Flutter + Dart
- Android host platform: Flutter Android embedding
- Backend/API (planned): Node.js/Express
- Database (planned): Supabase/PostgreSQL

## Project Structure

- lib: Dart application code
- test: Flutter widget/unit tests
- android: Android host project generated by Flutter
- pdfs: LaTeX presentation and report materials

## Getting Started

1. Ensure Flutter SDK is installed and on PATH.
2. Install dependencies:
   flutter pub get
3. Run the app:
   flutter run

## Notes About Migration

- Legacy native Android module files from the previous Kotlin setup were removed.
- The repository now uses Flutter tooling as the primary build and run path.

## Presentation Material

The LaTeX files in the pdfs folder are kept for project documentation and presentation output.
